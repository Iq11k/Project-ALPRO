LibrarianZ

LibrarianZ adalah sebuah proyek Mata Kuliah Algoritma dan Pemrograman kelompok kami. Proyek ini bertujuan untuk mengimplementasikan algoritma sorting dan searching menggunakan bahasa pemrograman Javascript dengan GUI menggunakan HTML & CSS. Tujuannya adalah untuk menunjukkan kemampuan dalam memahami dan menerapkan konsep-konsep dalam mata kuliah Algoritma dan Pemrograman.

Petunjuk Penggunaan :

1. Buka link https://librarianz.vercel.app/
2. Pada tampilan awal klik (kelola database ->) user akan di arahkan ke tampilan dashboard
3. Untuk menambahkan data buku klik button (Tambah data Buku) user akan di arahkan untuk mengisi detail data buku
4. Untuk mencari data buku user dapat menggunakan kolom cari buku di kanan atas
5. (jika sudah memiliki data buku) klik edit untuk mengubah data buku dan delete untuk menghapus data buku
6. (jika sudah memiliki data buku) pada kolom Tahun, Jumlah Halaman, dan Stok terdapat 2 button untuk sorting baik secara ascending maupun descending (sesuai icon)

Daftar algoritma yang diimplementasikan:

1. Selection Sort => digunakan untuk sorting data buku
2. Sequential Search => digunakan untuk mencari data buku
